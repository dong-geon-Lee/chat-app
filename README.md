## :mag_right: &nbsp; Intro
> 채팅 서비스 앱입니다.  
1:1 채팅 및 그룹 채팅 기능을 제공합니다. 
<br/>

## :link: &nbsp; Guide
> 첫번쨰: 레포지토리를 다운로드합니다.  
> 두번쨰: 터미널에 npm install 입력합니다.   
> 세번쨰: 터미널에 npm run dev 입력합니다.
